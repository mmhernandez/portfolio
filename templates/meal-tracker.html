Flex Daily Meal Tracker

Python | Flask | MySQL | Javascript | API | Bootstrap | CSS | Jinja2 | HTML

This app was created for a family member, who began a diet plan with a local weightloss company. Unfortunately, the company did not offer any means for electronic meal tracking (and only provided customers with lined paper). Voila - a perfect project opportunity was born!

Application Deployment

This project was deployed to AWS in an Amazon EC2 instance. A domain was purchased from Amazon and connected to the EC2 instance using the Amazon Route53 service. An SSL certificate was also purchased; using the Load Balancer tool, traffic is routed to HTTPS and references the cert for a secure site experience for users.
